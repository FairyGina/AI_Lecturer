# AI_Lecturer
PPT 파일을 주면 이미지 슬라이드를 기반으로 강의를 만들어주는 생성형 AI프로젝트입니다.

pptx: 해당 프로젝트에 대한 발표 ppt입니다.
ipynb 1, 2: 해당 프로젝트의 코드입니다.

주요 기능 및 서비스
1. PPT 슬라이드 기반으로 각 언어별로 슬라이드의 언어를 바꿔 재구성합니다. 또한 슬라이드의 정보(텍스트, 이미지, 표)에 따라 AI기반으로 생성된 발표 대본을 읽는 음성 또한 언어마다 다른 음성을 제공합니다.
2. PPT 전체 요약을 강의 끝에 정리하고, 강의 / 요약 으로 총 2가지 리포트가 AI 기반으로 생성됩니다. 이는 강의를 들으며 참고하게 글로 정리된 PDf를 강의 전체를 설명하는 리포트와, 강의를 요약한 리포트로 총 2가지로 나눠 제공됩니다.
